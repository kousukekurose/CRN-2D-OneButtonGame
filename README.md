# 2026-CRN-2D横スクロールアクションゲーム(条件：ボタン一つのみ)
2026年4月4日に公開予定の作品
『CRN-2D-OneButtonGame』 _readme 更新日｜2026.02.27

【作者：黒瀬 宏亮】

==============================================================================【ゲーム概要】

ボタンはひとつだけ。
ジャンプのみで攻略する2D横スクロールアクションゲームです。

敵やギミックをかわしながらゴールを目指します。
タイムアタック制のため、より速いクリアタイムを記録し、ランキング1位を目指しましょう。

==================================================================================【起動方法】

「CRN_2DSide-ScrollingGame.exe」を起動してください。

==================================================================================【操作方法】

| 操作        | ボタン   |
| --------- | ----- |
| 決定 / ジャンプ | 右クリック |

※本作はワンボタン操作で遊べます。

==================================================================================【基本ルール】

・ジャンプのみでステージを進行

・敵やギミックに触れるとミス

・ゴール到達でクリア

・クリアタイムがランキングに記録されます
